# 首爾同行 · Seoul Trip PWA

一個可安裝到手機主畫面、離線也能用的首爾旅遊導覽 App。

## 五個分頁
- **行程** — 5 天逐時程時間軸(做成地鐵路線圖風格),每個行程點可一鍵開地圖。
- **指南** — 景點／餐廳／夜生活／購物／保養精選卡片,可依類別篩選。
- **分帳** — 記錄共同開銷:誰先付、分給誰、韓幣或台幣;自動算出每人結餘與「最省步驟」的還款方式。結算幣別可切換。
- **匯率** — 韓幣⇄台幣即時換算(自動抓市場中價),含現金速查表;離線時用上次匯率。
- **須知** — 導遊叮嚀、航班／住宿、緊急電話(可直接撥號)、必備 App。

分帳資料存在手機瀏覽器本機(localStorage),不會上傳雲端;三個人各自記各自的裝置,或指定一台當公帳。

## 怎麼用 / 上線

### 最簡單:直接開來用
把整個資料夾傳到手機,用瀏覽器開 `index.html` 即可使用全部功能(分帳、匯率、行程都能用)。
> 註:用 `file://` 直接開時,無法「加到主畫面」變成獨立 App;要有這功能需下面的網址部署。

### 變成可安裝的 App(建議)
PWA 需要透過 **HTTPS 網址** 才能安裝。挑一個免費空間上傳這個資料夾:

- **Netlify Drop** — 打開 https://app.netlify.com/drop ,把整個 `seoul-trip` 資料夾拖進去,幾秒就得到一個網址。最快。
- **GitHub Pages** — 建 repo → 上傳檔案 → Settings → Pages → 選 main 分支。
- **Vercel / Cloudflare Pages** — 拖資料夾或連 GitHub 皆可。

上傳後打開那個網址:
- **iPhone(Safari)**:分享鈕 → 「加入主畫面」。
- **Android(Chrome)**:選單 → 「安裝應用程式 / 加到主畫面」。

之後從主畫面點開就是全螢幕 App,且離線可用(匯率需連網才會更新)。

## 檔案說明
```
index.html          主程式(所有畫面與邏輯都在這)
manifest.json       App 名稱、圖示、主題色
service-worker.js   離線快取
icons/              App 圖示(192 / 512 / maskable)
```

## 想改內容?
行程與景點資料都在 `index.html` 裡的 `DAYS`、`PLACES`、`TIPS` 幾個陣列,照著格式增刪即可,不用動樣式。
